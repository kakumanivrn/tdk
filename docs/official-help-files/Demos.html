Introduction

The Tcl Dev Kit provides a number of demonstration applications to help you get started with the TDK. In addition to the Tcl demos included with ActiveTcl, the following demos can be found in the demos directory beneath your Tcl installation directory, or, on Windows, in the Tcl Dev Kit Demos menu item in the ActiveState Tcl Dev Kit program group.

Top

Debugger

Coverage Testsuites

The Coverage Testsuites demonstrate how to use the Tcl Dev Kit Debugger to ensure that a tcltest package for a Tcl script library covers all paths through the code. These demos are described in detail in the Coverage Analysis Tutorial.

To access the demo:

  • Windows Start Menu: Start | Programs | ActiveState Tcl Dev Kit | Tcl Dev Kit Demos | Profiling and Coverage |
  • Windows Path: <install_dir>\demos\TclDevKit\TclDebugger\Coverage\
  • Unix Path: <install_dir>/demos/TclDevKit/TclDebugger/Coverage/

Hotspot Multiplication

The Hotspot Multiplication demos show how to use the Tcl Dev Kit Debugger's "profiling" feature to analyze the efficiency of your code. These demos are described in detail in the Hotspot Analysis Tutorial.

To access the demo:

  • Windows Start Menu: Start | Programs | ActiveState Tcl Dev Kit | Tcl Dev Kit Demos | Profiling and Coverage |
  • Windows Path: <install_dir>\demos\TclDevKit\TclDebugger\Profiling\
  • Unix Path: <install_dir>/demos/TclDevKit/TclDebugger/Profiling/

Debugging Subprocesses

The Debugging Subprocesses demo shows how to use the Tcl Dev Kit Debugger to spawn and debug subprocesses. This demo is described in detail in the Debugging Subprocesses Tutorial.

To access the demo:

  • Windows Start Menu: Start | Programs | ActiveState Tcl Dev Kit | Tcl Dev Kit Demos | Debugging Subprocesses
  • Windows Path: <install_dir>\demos\TclDevKit\TclDebugger\Spawning\
  • Unix Path: <install_dir>/demos/TclDevKit/TclDebugger/Spawning/

See also:

  • User Guide: Debugger

TclApp

BWidget

The BWidget demo shows how to use the BWidgets package with the Tcl Dev Kit's TclApp tool to generate a stand-alone program that contains all the files needed to run an application.

To access the demo:

  • Windows Start Menu: Start | Programs | ActiveState Tcl Dev Kit | Tcl Dev Kit Demos | Wrapping | BWidget Demo |
  • Windows Path: <install_dir>\demos\TclDevKit\TclApp\BWidgets\
  • Unix Path: <install_dir>/demos/TclDevKit/TclApp/BWidgets/

TclDom

The TclDom demos show how to "wrap" an application containing a package that relies on external DLLs (that are not stub-enabled).

To access the demo:

  • Windows Start Menu: Start | Programs | ActiveState Tcl Dev Kit | Tcl Dev Kit Demos | Wrapping | Advanced |
  • Windows Path: <install_dir>\demos\TclDevKit\TclApp\Tcldom\
  • Unix Path: <install_dir>/demos/TclDevKit/TclApp/Tcldom/

Expect Weather

The Expect Weather demo (Unix only) retrieves weather forecasts.

To access the demo:

  • Unix Path: <install_dir>/demos/TclDevKit/TclApp/Expect/

Img

The Img demo uses TclApp to generate a Starpack.

To access the demo:

  • Windows Start Menu: Start | Programs | ActiveState Tcl Dev Kit | Tcl Dev Kit Demos | Wrapping | Img Demo -- As Starpack
  • Windows Path: <install_dir>\demos\TclDevKit\TclApp\Img\
  • Unix Path: <install_dir>/demos/TclDevKit/TclApp/Img/

Klondike

The Klondike demo (a solitaire-type card game) uses TclApp to generate a Starpack.

To access the demo:

  • Windows Start Menu: Start | Programs | ActiveState Tcl Dev Kit | Tcl Dev Kit Demos | Wrapping | Klondike -- As Starpack
  • Windows Path: <install_dir>\demos\TclDevKit\TclApp\Klondike\
  • Unix Path: <install_dir>/demos/TclDevKit/TclApp/Klondike/

Tkcon

The Tkcon demo builds a standard console by using TclApp to "wrap" the portable Tkcon functions. The Tkcon console functions on all platforms where Tcl/Tk is available.

To access the demo:

  • Windows Start Menu: Start | Programs | ActiveState Tcl Dev Kit | Tcl Dev Kit Demos | Wrapping | Tkcon -- As Starkit
  • Windows Path: <install_dir>\demos\TclDevKit\TclApp\Tkcon\
  • Unix Path: <install_dir>/demos/TclDevKit/TclApp/Tkcon/

Miscellaneous

Miscellaneous TclApp demos include the hexplode game, the doctools processor and tkchat. All are available from the ActiveState Tcl Dev Kit | Tcl Dev Kit Demos | Wrapping Windows menu, or from the demos/TclDevKit/TclApp/TclApps directory.

Service Manager

The TclSvc Register ExprServer Service Manager demo (Windows only) installs a Windows service on a target system. The TclApp component of the Tcl Dev Kit is used to create Tcl applications that will be deployed as services. The Service Manager component is used to install the service on a target system (or to remove a Tcl service that was previously installed).

To access the demo:

  • Windows Start Menu: Start | Programs | ActiveState Tcl Dev Kit | Tcl Dev Kit Demos | TclSvc Register ExprServer
  • Windows Path: <install_dir>\demos\TclSvc\
  • Windows Path: <install_dir>\demos\TclDevKit\TclApp\Tkcon\
  • Unix Path: <install_dir>/demos/TclDevKit/TclApp/Tkcon/

See also:

    • TclApp
    • Service Manager

Virtual Filesystem Explorer

The Explore Starkit and Explore Starpack demos both make use of the Virtual Filesystem Explorer (VFSE). Before you can run these demos, you must generate the associated starkits and starpacks by running the TclApp BWidgets demo.

To access the demos:

  • Windows Start Menu: Start | Programs | ActiveState Tcl Dev Kit | Tcl Dev Kit Demos | Wrapping | BWidget Demo | Explore Starkit


    • and

    Start | Programs | ActiveState Tcl Dev Kit | Tcl Dev Kit Demos | Wrapping | BWidget Demo | Explore Starpack

    Alternatively, run the executables created in the Building TclApps Tutorial from the command line by launching them from the directory in which the executables were created.

  • Windows Path for Starkits: <install_dir>\bwdemo.tcl
  • Unix Path for Starkits: <install_dir>/bwdemo.tcl
  • Windows Path for Starpacks: <install_dir>\bwdemo.exe
  • Unix Path for Starpacks: <install_dir>/bwdemo.exe

See also:

  • User Guide: Virtual Filesystem Explorer

Cross Reference Tool

The Xref BWidgets demo uses the Cross Reference Tool to extract component information (such as packages, namespaces, commands, etc) from the BWidgets package. (See also the TclApp BWidgets demo.)

To access the demo:

  • Windows Start Menu: Start | Programs | ActiveState Tcl Dev Kit | Tcl Dev Kit Demos | Xref BWidgets
  • Windows Path: <install_dir>\bin\tclxref.tcl -tap <install_dir>\lib\bwidget1.6\bwidget.tap
  • Unix Path: <install_dir>/bin/tclxref -tap <install_dir>/lib/bwidget1.6/bwidget.tap

See also:
